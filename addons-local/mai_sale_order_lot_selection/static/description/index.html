Sale Order Lot Selection(Community & Enterprise)

    Key Features
  • Using This module add lot number in sale order line.
  • FIFO method is used for lot Number.
  • Tested on the Odoo.sh.
  • Compatible for community and enterprise version of odoo.

Sale Order Lot Selection


For using this module first enable Lots and Serial numbers from Inventory -> Configuration -> Lots Serial Numbers.
Create a Sales order and add lot number as per the screen.And for select following configuration from product's Inventory tab-> Traceability -> By lots.

Check related delivery order and based on that click on validate button.

Here you can see delivery order is done with qty done qty.

Once you click on register lots button you can see the details of the operation with lot and done qty.
